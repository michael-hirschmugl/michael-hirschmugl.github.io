Interesting Projects

  • SNES-Drone
    A creative hardware project turning the Super Nintendo's sound chip (SPC700) into a drone synthesizer using a custom cartridge.
  • PS2 BlueRetro
    A hardware mod to add Bluetooth gamepad support to the PlayStation 2 using an ESP32-based adapter.
  • SNESDRONE
    A project to control the SNES sound chip via custom cartridges for real-time music synthesis.
  • open-hw-cnn
    An open-source hardware accelerator framework for convolutional neural networks on FPGAs.
  • sun_follower
    A FreeRTOS-based embedded firmware project to track and follow sunlight using motorized hardware.
